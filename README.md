# SOCIAL_MEDIA_TOP     
## Alex Srebernic    

Hello, in this "clone" of Facebook where you can:
- Create a user 
- Edit your profile 
- Add friends
- Chat in real time with your friends 
- Create posts
- Like posts
- Dislike posts 

Currently realtime interaction not working because netlify don't support websockets, but believe me it works.
